
# WarehouseMind AI

WarehouseMind AI is a multi-agent warehouse operations control tower built using FastAPI, Google ADK, and Gemini.

## Tech Stack

- Python
- FastAPI
- SQLite (development)
- SQLAlchemy
- Docker
- Google ADK (coming in later phases)
- Gemini
